HTML

  • The head element contains information about the webpage.
  • The body element contains the content of the website visible to the user.
  • The h1-h6 elements define the level of the heading, should only be one h1 per page.
  • The ol, ul, li elements are the ordered lists, unorder lists, and list items used to create different lists.
  • The img element contains information about the images shown on a website.
  • The br element designates a line break between blocks of content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A . before an element designates a class selector, any HTML element assigned to that class will gain those properties from CSS
  • box-shadow adds a shadow effect around an elements frame
  • Multiple effects can be separated by commas.
  • class= assigns the CSS class to an element.

Git

  • cd is used to change the current directory. Example: "cd bootcamp" changes to the bootcamp directory
  • git status is used to check branch we are working with.
  • git checkout -b branch-name creates a new branch and switches to it
  • git pull origin main fetches commits from the main branch
